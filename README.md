# Spots
 This is the third project of the Software Enginnering Program at Triple Ten. 

## Project Description

Spots is a new social media platform desgined for Photographers and Tourists. It can be used to share unique locations, portfolios, and to even connect with others that share the same passion. 

## Project Features 

- HTML
- CSS
- Responsive Design 
- Grid & Flexbox
